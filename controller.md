# Gamepad Controls (Xbox & PlayStation)

Berikut adalah daftar lengkap pemetaan tombol untuk Gamepad (Xbox dan PlayStation) yang digunakan dalam prototype game ini:

### Movement & Actions
- **Left Stick** : Bergerak (Move)
- **Right Stick** : Menggerakkan Kamera (Look)
- **A / Cross** : Melompat (Jump)
- **L3 (Klik Analog Kiri)** : Berlari (Sprint)
- **B / Circle** : Berjongkok (Crouch) / Meluncur (Slide saat berlari)
- **Y / Triangle** : Melayang (Glide - saat berada di udara)

### Combat & Items
- **R1 / RB** : Menembak (Shoot)
- **L1 / LB (Tahan)** : Membidik (Aim / Zoom)
- **Y / Triangle** : Mengganti Senjata (Switch Weapon - ke senjata berikutnya)
- **D-Pad Atas (Up)** : Menyalakan/Mematikan Senter (Toggle Flashlight - jika sudah diambil)

### Interactions & UI
- **X / Square** : Berinteraksi (Mengambil Item, Meretas CCTV, Menekan Keypad)
- **X / Square** *(Di dalam CCTV)* : Beralih ke kamera selanjutnya di ruangan yang sama
- **D-Pad Bawah (Down)** : Menginspeksi Dokumen (Inspect Mode - untuk melihat item dokumen)
- **B / Circle** : Keluar dari mode CCTV / Keluar dari Inspect Mode / Menutup UI Popup
