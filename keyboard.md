# Keyboard & Mouse Controls

Berikut adalah daftar lengkap tombol input yang digunakan dalam prototype game ini:

### Movement & Actions
- **W, A, S, D** : Bergerak (Move)
- **Space** : Melompat (Jump)
- **Left Shift** : Berlari (Sprint)
- **Left Ctrl** : Berjongkok (Crouch) / Meluncur (Slide saat berlari)
- **F** : Melayang (Glide - saat berada di udara)

### Combat & Items
- **Left Click** : Menembak (Shoot)
- **Right Click (Tahan)** : Membidik (Aim / Zoom)
- **1 - 9 / Mouse Scroll** : Mengganti Senjata (Switch Weapon)
- **4** : Menyalakan/Mematikan Senter (Toggle Flashlight - jika sudah diambil)

### Interactions & UI
- **E** : Berinteraksi (Mengambil Item, Meretas CCTV, Menekan Keypad)
- **E** *(Di dalam CCTV)* : Beralih ke kamera selanjutnya di ruangan yang sama
- **I** : Menginspeksi Dokumen (Inspect Mode - untuk melihat item dokumen)
- **ESC** : Keluar dari mode CCTV / Keluar dari Inspect Mode / Menutup UI Popup
